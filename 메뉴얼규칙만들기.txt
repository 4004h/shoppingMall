3. 디렉토리 규칙
 3-1. js : js 디렉토리(폴더)는 주요 자바스크립트 외 jQuery 라이브러리 파일을 저장하는 장소

 3-2. css : css 디렉토리(폴더)는 주요 CSS3와 관련된 파일을 저장하는 장소

 3-3. img : img 디렉토리(폴더)는 웹 문서상에서 사용된 이미지를 저장하는 장소

단, 이미지 저작권을 감한하여, 공개용 혹은 비상업용 이미지 위주로 포트폴리오 작업시 활용

 3-4. font : *.ttf 로 끝나는  트루타입 글꼴 보다는 파일에 크기(용량)이 웹 문서에 적용하기 딱 좋은 글꼴타입인 *.eot / *.woff 형식을 저장하는 장소 활용.
단, 저작권을 감안하여 본 포트폴리오에는 비상업용인 공개용 폰트만 사용함.
